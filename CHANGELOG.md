# 鑫瓒进销存 V1.3 更新日志

## 🆕 新增功能

### 销售报价模块（全新）
- 报价单列表/新增/编辑/详情/打印/PDF导出，单号前缀 BJ
- 状态流转：草稿→转销售订单（锁定）→撤回（可重新编辑再转）
- 权限点 `sales_quote`，管理员看全部、销售经理看自己
- 详情页渲染"产品项目方案单"打印模板

### 产品项目方案单打印模板
- 新增打印模板类型 quote，含商品图片、名称、编码、描述、价格、数量、金额
- 自动初始化（首次访问时数据库不存在则创建，已存在跳过）
- `{total_amount_cn}` 自动转中文大写金额

## 🐛 Bug 修复

### 销售报价
- `quote_view.php` 重复声明函数导致 Fatal error
- 撤回后缺少"编辑"按钮——已增加编辑入口
- 转订单时 `warehouse_id` 硬编码为 0 导致后续出库失败——改为自动取首个可用仓库

### 出库
- 保存失败时显示具体错误原因（替代"请稍后重试"）

### 盘点管理
- 新产品不在 `inventory` 表中导致盘点不显示——改为 `products LEFT JOIN inventory`

### 角色权限
- 补上遗漏的 `sales_quote`（销售报价）权限项

### 打印模板
- `buildItemsHtml` 正则兼容带 `style` 属性的 `<th>`
- 浏览器打印时背景色丢失——增加 `print-color-adjust: exact`
- 描述列太窄导致行高过大——优化列宽分配和容器 padding
- 表头文字看不清——删除打印 CSS 中 `th` 背景色，保留模板内联样式
- 商品管理表单中描述与备注位置互换

## 🛠 数据库变更
- 新增 `sales_quotes`、`sales_quote_items` 表
- `products` 表新增 `description` 字段（TEXT）

## 📋 版本
- 当前版本：V1.3
- 发布日期：2026-07-18

# 历史版本（V1.2）

## 🆕 新增功能

### CRM客户关系管理模块（全新）
- **客户管理**：完整的客户CRUD，支持多维度搜索筛选、电话查重、自动编码
- **跟进记录**：6种跟进方式 + 4种结果状态 + 附件上传 + 三个Tab视图（全部/今日待跟进/今日已跟进）

- **首页看板**：CRM数据卡片 + 图表 + 待跟进列表直接嵌入首页

### 权限体系
- 新增9个CRM权限点，支持管理员/业务经理两级数据隔离

### 客户来源管理
- 支持自定义来源分类、排序、启用/禁用

- CSRF Token 输出安全加固
