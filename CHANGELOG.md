# 鑫瓒进销存 V1.2 更新日志

## 🆕 新增功能

### CRM客户关系管理模块（全新）
- **客户管理**：完整的客户CRUD，支持多维度搜索筛选、电话查重、自动编码
- **跟进记录**：6种跟进方式 + 4种结果状态 + 附件上传 + 三个Tab视图（全部/今日待跟进/今日已跟进）
- **公海池**：智能自动回收（闲置回收 + 认领未跟进回收）+ 认领/分配/转移/归池
- **数据报表**：6个核心指标 + 4种图表（Chart.js）+ 业务经理排名TOP10
- **Excel导出**：CSV格式，19列完整数据，Excel直接打开不乱码
- **首页看板**：CRM数据卡片 + 图表 + 待跟进列表直接嵌入首页

### 权限体系
- 新增9个CRM权限点，支持管理员/业务经理两级数据隔离

### 客户来源管理
- 支持自定义来源分类、排序、启用/禁用

## 🛠 安全修复
- CSRF Token 输出安全加固
- 用户/角色页面防XSS注入
- 售后追踪码密码改用hash存储
- SSL证书验证启用
- 移除硬编码API密钥
- 模板下载增加权限检查
